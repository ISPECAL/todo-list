#TO-DO LIST APP 
This app will serve the purpose of having a place besides your mind to put tasks that need to be done.

[!Note]
Installation instructions
Node.js MUST be installed
[Download here](https://nodejs.org)
1.You will need to clone the repository
2.Once cloned open the terminal inside the project directory and install the dependencies.
[!Important]
npm install
[!Important] 
npm run dev
This will start the server 
then open the local link shown in the terminal (which should look something like this http://localhost:5173/) to view the app in your browser.
